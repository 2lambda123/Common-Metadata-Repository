# authz

*An authorization utility library for CMR services*

[![][logo]][logo]


## About [&#x219F;](#contents)

This library wraps multiple calls to CMR and ECHO services for authentication
and token checking.


## Dependencies [&#x219F;](#contents)

* Java
* `lein`
