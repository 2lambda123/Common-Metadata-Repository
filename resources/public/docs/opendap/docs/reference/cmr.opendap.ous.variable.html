async-get-metadata

(async-get-metadata search-endpoint user-token {variable-ids :variables})

Given a ‘params’ data structure with a ‘:variables’ key (which may or may not have values) and a list of all collection variable-ids, return the metadata for the passed variables, if defined, and for all associated variables, if params does not contain any.

create-opendap-lookup

(create-opendap-lookup lon-lo lat-lo lon-hi lat-hi)

This lookup is needed for when latitude -90N is stored at the 0th index and 90N is stored at the highest index (whose actual number will varry, depending upon the resolution of the data).

create-opendap-lookup-reversed

(create-opendap-lookup-reversed lon-lo lat-lo lon-hi lat-hi)

This lookup is needed for when latitude 90N is stored at the 0th index and -90N is stored at the highest index (whose actual number will varry, depending upon the resolution of the data).
